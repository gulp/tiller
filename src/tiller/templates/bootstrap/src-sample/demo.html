Dark Mode Implementation

This standalone HTML file demonstrates a complete dark mode implementation with theme persistence. The design uses CSS variables for seamless theme switching and localStorage to remember your preference across sessions.

How It Works

The theme system detects your system preference on first visit using the prefers-color-scheme media query. When you toggle the theme, your choice is saved to localStorage and will persist on subsequent visits.

Key Features

System Preference Detection: Automatically matches your operating system's theme setting on first visit.

Persistent Storage: Your theme choice is saved locally and restored on page reload.

Smooth Transitions: All color changes animate smoothly using CSS transitions.

Implementation

The complete implementation uses CSS variables, local storage, and vanilla JavaScript. Here's the core theme-switching logic:

function toggleTheme() {
    const current = document.documentElement
        .getAttribute('data-theme');
    const next = current === 'dark' ? 'light' : 'dark';

    document.documentElement
        .setAttribute('data-theme', next);
    localStorage.setItem('theme', next);
}

CSS Variables

Theme colors are defined using CSS custom properties, allowing instant theme switching without JavaScript manipulation of individual styles:

:root {
    --color-bg: #fdfbf7;
    --color-text: #1a1815;
    --color-accent: #2563eb;
}

[data-theme="dark"] {
    --color-bg: #0f1419;
    --color-text: #e4e1db;
    --color-accent: #5b9dff;
}
